Welcome to

U.S Accidents!

Presented by Teja, Simon, and Aaron

This project will cover U.S Accident data. Various views will be avaliable, allowing the user to sort, show all, filter, summarize, and limit the visible rows of data. In the future, data visualization will be an important part of our project. We will create a heat map that shows crashes per region per month, and the ability to move around in time, seeing the heat map adjust dynamically. We will also allow the user to adjust the data to simulate changes in accident trends, adding, deleting, or updating data.
Our data is from 2016-2023 and sourced from Kaggle: us-accidents/data
Listed below is the json format for each tuple, each tuple will have 46 columns/variables.
