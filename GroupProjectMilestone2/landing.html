Milestone 2 - Project Dashboard

By Aaron Quashnock, Simon Little, Teja Desari

This dashboard provides various views of sample hardcoded data that will be supplemented with the full data later.

Show All

Show all of the available data in the table.

Filter

Filters table rows to only include those with a specified value in a specified value.

=

Sort

Sort the order of the rows based on the values within a column. This is done lexicographically for string values and numerically for numbers.

Limit Viewable Columns

Limit the total number of columns shown in the table.


Summary

Displays a list of all unique values in each column.
